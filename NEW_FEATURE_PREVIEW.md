## 🎬 Demo Commands Available

### 1. `/scan` - Wallet Scanner
Shows 3 top-performing wallets with:
- Wallet addresses
- Win rates (70-78%)
- Average ROI
- Recent activity

### 2. `/watchlist` - Monitored Traders
Displays:
- 5 tracked traders
- Online status
- Last trade info
- Current tokens

### 3. `/portfolio` - Current Positions
Shows:
- Total portfolio value
- Today's P&L
- 3 open positions (PEPE, WIF, BONK)
- Entry/current prices
- Profit percentages

### 4. `/analyze PEPE` - Token Analysis
Displays:
- Safety score (87/100)
- Security checks
- Market data
- AI recommendation
